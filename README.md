AutoXiaomi
==========

APIs for buying Xiaomi-Tw Product


How to use ?
------------

1)

  edit the buy.config
  
  write your info in it
  

2)
  
  edit \_\_main\_\_.py
  
  add your account & pwd
  
  use python to run
      

P.S
-------
  
  buy_method1 function can only apply on the order without captcha.
  
  itemid can be found at Xiaomi's website.
